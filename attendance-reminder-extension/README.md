# 授業出席リマインダー Chrome拡張機能

東京工科大学のサイト（https://service.cloud.teu.ac.jp/）で、各授業の開始時間に出席登録を促すバナーを表示するChrome拡張機能です。

## 機能

- 授業開始10分前から20分後までの30分間、ページ上部にリマインダーバナーを表示
- ×ボタンでバナーを閉じることができます
- 一度閉じたバナーは同じセッション内で再表示されません
- 5つの授業時間に対応

### 対応時間帯

| 時限 | 開始時刻 | 表示時間帯 |
|------|---------|-----------|
| 1限 | 8:50 | 8:40-9:10 |
| 2限 | 10:45 | 10:35-11:05 |
| 3限 | 13:15 | 13:05-13:35 |
| 4限 | 15:10 | 15:00-15:30 |
| 5限 | 17:05 | 16:55-17:25 |

## インストール方法

1. Chromeブラウザで `chrome://extensions/` を開きます
2. 右上の「デベロッパーモード」を有効にします
3. 「パッケージ化されていない拡張機能を読み込む」をクリック
4. この `attendance-reminder-extension` フォルダを選択します

## 使い方

1. 拡張機能をインストールすると、自動的に動作します
2. https://service.cloud.teu.ac.jp/ にアクセスすると、該当時間帯にバナーが表示されます
3. バナーの×ボタンをクリックすると閉じることができます

## ファイル構成

```
attendance-reminder-extension/
├── manifest.json          # 拡張機能の設定ファイル
├── content.js            # バナー表示ロジック
├── styles.css            # バナーのスタイル
├── icons/
│   ├── icon16.png       # 16x16 アイコン
│   ├── icon48.png       # 48x48 アイコン
│   └── icon128.png      # 128x128 アイコン
└── README.md            # このファイル
```

## カスタマイズ

時間帯を変更したい場合は、`content.js` の `CLASS_PERIODS` 配列を編集してください。

```javascript
const CLASS_PERIODS = [
  { period: 1, startHour: 8, startMinute: 40, endHour: 9, endMinute: 10, label: '1限' },
  // ...他の時限
];
```

## 注意事項

- この拡張機能は https://service.cloud.teu.ac.jp/ でのみ動作します
- ブラウザを再起動するまで、閉じたバナーの記録が保持されます
